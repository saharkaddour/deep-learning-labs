# deep-learning-labs

This repository contains practical notebooks related to machine learning and deep learning using Python and TensorFlow.

The goal of this repository is to experiment with different machine learning techniques such as optimization methods and convolutional neural networks for image classification.

## Projects

### 1. CNN Image Classification (CIFAR-10)

This notebook implements a Convolutional Neural Network (CNN) to classify images from the CIFAR-10 dataset.

Main steps:
- Dataset loading and preprocessing
- Data normalization
- CNN architecture design
- Model training and validation
- Performance visualization (loss and accuracy curves)
- Confusion matrix and classification report

Technologies used:
- TensorFlow / Keras
- NumPy
- Matplotlib
- Scikit-learn

### 2. H2 Optimization

This notebook explores optimization techniques applied to machine learning problems.

Main steps:
- Problem formulation
- Optimization approach implementation
- Training and evaluation of the model
- Performance visualization

## Tools

- Python
- TensorFlow
- Google Colab
- Matplotlib
- Scikit-learn

## How to Run

1. Clone the repository

2. Open the notebooks in **Google Colab** or **Jupyter Notebook**

3. Run the cells sequentially.
